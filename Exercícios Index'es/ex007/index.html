Textos deletados

Podemos marcar um texto como riscado, para esclarecer que ele deve ser lido mas não considerado utilizando a tag <del>.

Textos inseridos

Podemos marcar um texto como inserido, para dar ênfase e indicar que foi adicionado depois.

Textos super-escritos

Podemos fazer um texto super-escrito para demonstrar uma equação como por exemplo x20+3

Textos sub-escritos

Podemos fazer um texto sub-escrito para demonstrar uma fórmula química como por exempo H2O
